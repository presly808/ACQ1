Инструкция по созданию джобов в Jenkins

Устанавливаем Гит плагин
1. Заходим в http://localhost:8080/
2. Нажать "Настроить Jenkins" => спарва в появившемся списке выбираем "Управление плагинами"
3. На странице управления плагинами переходим на вкладку "Доступные", ищем GitHub Plugin и устанавливаем


Конфигурируем Jenkins
1. На главной странице Нажать "Настроить Jenkins" => спарва в появившемся списке выбираем "Конфигурирование системы"
2. В секции "JDK" убираем галочку "Install automatically" и заполняем "Имя" и "JAVA_HOME" (у меня Имя - Java; JAVA_HOME - C:\Program Files\Java\jdk1.8.0_60)
3. Прописать на машине Path C:\Program Files (x86)\Git\cmd, тогда секцию "Git" не трогаем (по дефолту у меня там значения Name - Default; Path to Git executable - git.exe)
4. В секции "Maven" указываем "Имя" и "MAVEN_HOME"
5. Нажимаем кнопку "Применить"

Создаем и настраиваем джобу
1. Нажимаем кнопку "Создать Item"
2. В поле "GitHub project" копируем линку с гита(у меня - https://github.com/ViTaLESs/ACQ1/)
3. В секции "Управление исходным кодом" выбираем Git и в "Repository URL" копируем линку с гита(у меня - https://github.com/ViTaLESs/ACQ1/)
3. В секции "Сборка" пишем в поле "Корневой POM" - pom.xml; "Goals and options" - test
4. Нажимаем кнопку "Применить"

Все, теперь джобу можно запускай)
